# Advocate Portfolio Website

A modern, responsive, and SEO-friendly portfolio website for Advocates/Lawyers built with React, Vite, and Tailwind CSS. Designed to showcase legal services, experience, case studies, client testimonials, and provide an easy way for clients to book consultations.

## Demo

Live Website: https://advocate-portfolio-demo-opal.vercel.app/

## Tech Stack

- React 19
- Vite 8
- React Router DOM
- Tailwind CSS v4
- Framer Motion
- React Helmet Async
- Lucide React
- EmailJS

## About the Project

This portfolio website helps advocates and law firms establish a professional online presence. It includes:

- Professional Landing Page
- About Section
- Practice Areas
- Case Studies
- Career Timeline
- Client Testimonials
- Blog with Search & Categories
- FAQ Section
- Contact Form
- Consultation Booking
- SEO Optimization
- Fully Responsive Design
